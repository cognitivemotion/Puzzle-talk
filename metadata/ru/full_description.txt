Escapepod - это проигрыватель подкастов с минималистским подходом, который может не понравиться всем. Приложение имеет только один экран. Элементы управления воспроизведением и список подкастов с пятью последними эпизодами. У Escapepod нет функции обнаружения подкастов. Он предлагает только очень простой параметр поиска и открывает ссылки на подкасты RSS, когда вы нажимаете на них в веб-браузере.


ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ

У ESCAPEPOD ЕСТЬ ОЧЕРЕДЬ ВОСПРОИЗВЕДЕНИЯ?
Escape Pod обладает простой следующей функцией. Нажмите «Воспроизвести» в эпизоде, одновременно слушая другой. Вам будет предоставлена возможность добавить его в следующий слот.

КАКОВЫ НАСТРОЙКИ ПО УМОЛЧАНИЮ?
- Автообновление не загружает файлы по сотовой сети
- Escapepod по умолчанию сохраняет только два эпизода

МОЖЕТ ЛИ ESCAPEPOD ХРАНИТЬ БОЛЬШЕ ДВУХ ЭПИЗОДОВ?
Escapepod пытается уменьшить количество эпизодов, которые он хранит. Вот правила для этого:
- Escapepod хранит последние два эпизода.
- Эпизоды, которые были запущены, или которые были загружены вручную, тоже хранятся.
- Максимальное количество доступных (и сохраненных) эпизодов - пять

ПОДДЕРЖИВАЕТ ЛИ ESCAPEPOD OPML?
Вы можете импортировать список подкастов, используя формат обмена OPML. Просто откройте файл OPML с помощью файлового менеджера и выберите Escapepod в качестве обработчика. Текущая коллекция подкаста может быть экспортирована через Settings. Кроме того, Escapepod хранит актуальный файл OPML в папке /Android/data/org.y20k.escapepod/files/collection/.

Я НЕ МОГУ ДОБАВИТЬ ПОДКАСТ Х. ПОЧЕМУ?
Пожалуйста, сообщайте о любых подкастных каналах, которые в настоящее время не работают, с помощью шаблона GitHub Issue "Feed Issue" (https://github.com/y20k/escapepod/issues/). Примечание: Escapepod не поддерживает видеоподкасты.

ОТКУДА БЕРУТСЯ РЕЗУЛЬТАТЫ ПОИСКА ПОДКАСТОВ?
Escapepod ищет в онлайн базе данных gpodder.net (https://gpodder.net/directory/). Gpodder.net - отличная страница, если вы ищете что-то новое для прослушивания.
